# 生物馆研究简报

## 阶段边界

MUSEUM-00 只建立研究问题、来源风险和扩展模型，不制作生物馆、动画或公开数据集。

## 未来探索路径

`地球 → 生物地理界/生态区 → 典型栖息地 → 生态系统 → 食物网与物质循环 → 动植物/真菌/微生物 → 物种详情与行为场景`

路径不是单向层级真理：物种跨生态区，生态系统边界具有尺度和模型，关系随季节/生命阶段/地点变化。用户可从物种反向进入栖息地和网络。

## 研究问题

- 采用何种生态区框架及版本，如何表达边界不确定性？
- taxon identity 如何对齐 COL、GBIF、EOL 与馆藏标本并保留冲突？
- occurrence、推定分布、稳定栖息地和保护评估如何分离？
- GloBI 关系如何回溯原始研究并确定行为适用范围？
- IUCN 专门再发布条款下哪些字段/图层只能链接或需授权？
- 真实影像、科学重建、机制动画和假说如何在视觉上区分？

## 原则

不只用线性食物链；关系包括捕食、竞争、互利、偏利、寄生、传粉、分解和物质循环。动画先有 Evidence，再做脚本；单次观测不证明长期分布；保护状态必须记录评估版本、日期、范围和来源。

## MUSEUM-BIO-00 预期输出

条款复核、源适配器样例、生态区选择 ADR、taxon/occurrence/distribution schema、少量非发布 fixtures、行为证据 rubric、IUCN 授权判断和目标物种/生态系统选择标准。未通过权利与科学审核前不产生公开动画。
