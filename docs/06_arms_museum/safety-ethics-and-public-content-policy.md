# 武器博物馆安全、伦理与公开内容政策

## 适用范围与默认状态

本政策适用于未来武器博物馆的文字、数据、关系、媒体、3D、互动、搜索、推荐、外链、AI 草稿和儿童/家庭路径。`MUSEUM-02A` 的默认状态是：仅有不可点击的门户卡片，不采集正式器物，不下载媒体，不开放路由，不展示现代操作资料或图像化伤害。

公开内容必须同时通过来源、Claim–Evidence–Source、学科、权利、安全、年龄适宜性和发布审核。公开可访问不等于适合再发布；元数据许可不继承给媒体。

## 可以公开的内容

在上述门禁通过后，可以公开：

- 历史背景、年代、地点和具体语境；
- 博物馆器物的高层次描述、公开馆藏编号与来源；
- 材料和一般工艺传统，不含可执行参数；
- 高层次机械或物理原理，不含操作、制造或优化路径；
- 甲胄、防护和器物之间的历史关系；
- 保存、腐蚀、材料老化和博物馆保护知识；
- 仪式、象征、法律、制度和社会史；
- 收藏史、来源史、掠夺、交易、出土、归还与争议；
- 冲突对平民与社会的影响；
- 裁军、限制、退出使用、转化和纪念历史；
- 非操作性的 3D 观察与高层次部件名称。

## 不得公开为教学、互动或推荐的内容

不得提供：

- 制造武器、弹药、爆炸物或关键部件的逐步流程；
- 可执行尺寸、容差、配方、模具、加工参数、工程图或可直接复现的 schematics；
- 改装、恢复功能、增强威力、提高射速、精度、隐蔽性或可靠性的方法；
- 装填、发射、瞄准、拆装、故障排除、维修启用或战术使用教程；
- 绕过安全装置、监管、追踪、检测或访问控制的方法；
- 采购、价格、库存、供应商、交易渠道、购买链接或规避限制的信息；
- 对人体弱点、伤害效率、致命效果或穿透效果的优化比较；
- 面向现实目标的战术模拟、训练或任务建议；
- 爆炸物、燃烧物或化学武器的配方、制作和部署细节；
- “最致命”“最佳攻击”“最佳目标”“最适合……”等游戏化指标；
- 对极端主义、恐怖主义、战争暴行或压迫的庆祝性、美化性呈现。

内容即使来自公开网页、档案或技术手册，也不能因“可访问”而绕过本政策。

## 现代和受监管武器

对现代、仍在服役或受监管的武器：

- 只作高层次历史、政策、设计史、法律与博物馆说明；
- 不发布操作 schematics、维修手册、训练资料或详细操作部件视图；
- 不链接商业销售、交易或库存页面；
- 不提供采购、比较、选择或性能优化建议；
- 不把公开技术资料自动视为适合本馆再发布；
- 必须由学科审核者、安全审核者和发布审核者共同批准，权利审核另行独立完成。

年代范围、现代武器边界和具体审核等级由 `OD-011` 保持未决，不能在实现中静默关闭。

## 敏感内容与年龄适宜性

未来内容必须支持：

- 暴力、战争、死亡、殖民、奴役、种族迫害和极端主义相关提示；
- 默认不展示图像化伤害，不用惊吓性缩略图吸引点击；
- 儿童/家庭路径使用非图像化、非操作化解释；
- 访客可跳过敏感材料，跳过不会阻断基本学习路径；
- 文字替代不得比视觉内容更具伤害性；
- 历史仇恨或极端主义符号只在明确批判、解释和来源语境中呈现；
- 纪念、伤亡和平民影响与器物说明建立可见联系，不把器物从人类后果中抽离。

## 内容与互动审核

每个候选内容至少记录：内容类型、受众、年代、敏感主题、操作化风险、现代/受监管状态、来源与证据、metadata/media 权利、审核者、审核日期、状态历史和撤回触发。

发布前必须逐项检查：

1. 是否只陈述经来源支持的历史与博物馆事实；
2. 是否包含可执行步骤、参数、采购、优化或现实使用建议；
3. 是否美化暴力、战争、殖民、压迫或极端主义；
4. 是否为儿童/家庭与一般公众提供适当提示和非图像化路径；
5. 是否保留来源争议、反证、文化财产与归还语境；
6. 是否提供键盘、文字替代、减少动画和非颜色编码；
7. 是否通过 public input 与 production artifact 安全扫描。

AI 只能产生候选或草稿，不能补全被本政策禁止的细节，也不能成为事实的唯一来源。

## 事故、投诉与撤回

发现操作化内容、权利问题、错误语境、图像化伤害误发布或极端主义美化风险时，应立即阻断构建或撤下受影响记录，保留最小审计信息，发布新版本移除公开字节/引用，并记录原因、范围、时间、负责人和恢复条件。恢复需要新的学科、安全、年龄适宜性、权利与发布审核。
