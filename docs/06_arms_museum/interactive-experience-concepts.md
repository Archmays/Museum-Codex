# 武器博物馆互动体验概念

## 状态

以下均为未来概念，不是已实现功能，不创建路由、数据、3D、模拟器或媒体。本阶段只定义学习目的、安全边界和无障碍要求。

## 优先候选

| 概念 | 学习问题 | 安全与策展边界 |
|---|---|---|
| 器物时空图 | 形制、材料和术语如何随时间与地区变化？ | 不用相似性推断直接影响；现代边界不永久投射到历史 |
| 攻防共演化 | 器物与防护如何在具体证据和制度中共同变化？ | 只作历史解释，不给现实优化、穿透或对抗建议 |
| 材料实验室 | 青铜、铁、钢、木、皮革和复合材料的一般性质有何差异？ | 只比较一般性质、老化与保护，不提供配方、热处理或加工参数 |
| 保存显微镜 | 锈蚀、裂纹、修复和保护如何改变我们看到的器物？ | 使用非操作性博物馆保护说明，不教授恢复武器功能 |
| 来源旅程 | 器物如何经历制造、使用、收藏、转移、出土、掠夺和归还？ | 保留争议与反证，不把所有权链伪装为无缺口事实 |
| 仪式与实战 | 礼仪、象征、表演和实际历史使用如何区分？ | 不通过动作模拟教授使用；分类必须由来源支持 |
| 神话与证据 | 影视、游戏和大众叙事中的说法能否被证据支持？ | 不重演伤害或把纠错变成性能排行榜 |
| 同类器物比较 | 不同时代、地区、材料和策展体系如何描述相似器物？ | 不显示“最佳”“最致命”或现实选择建议 |
| 人类代价路径 | 器物如何连接战争、平民、殖民、压迫和记忆？ | 默认非图像化；敏感内容提示、跳过和儿童路径齐备 |
| 裁军与转化 | 武器如何退出使用、被销毁、转化、纪念或进入博物馆？ | 不展示可逆恢复、拆装或规避监管步骤 |

## 统一禁止项

互动不得：

- 模拟对人的伤害、人体弱点、命中、穿透、致命性或现实目标；
- 训练制造、装填、发射、瞄准、拆装、维修启用、改装或战术使用；
- 提供供应商、价格、库存、购买链接、法规规避或性能选择建议；
- 使用击杀、准星、子弹、爆炸、血迹、军事 HUD 或排行榜奖励机制；
- 把极端主义、战争暴行、殖民或压迫变成庆祝性任务；
- 隐藏不确定性、反证、来源争议或人类影响。

## 无障碍与家庭路径

- 每个图、时间轴、3D 或空间界面提供等价的文本/表格路径；
- 键盘、触控、读屏、200% 缩放、减少动画、forced colors 和低带宽模式可完成同一学习任务；
- 装饰图形 `aria-hidden=true`，状态与风险提示不只靠颜色；
- 敏感内容可跳过，默认不自动播放声音、动画或图像化材料；
- 儿童/家庭路径减少术语和同时可见信息，但引用同一 Claim，不生成较“简单”却互相冲突的事实；
- 3D 观察默认限制为非操作性的外观、材料、保存状态和高层次部件名称。

## 未来实现门槛

每个概念只有在 `MUSEUM-ARMS-00` 完成、用户另行授权、所需数据 publishable、媒体权利闭合、安全/年龄审核通过、文字替代完成且真实设备 QA 通过后才能进入原型。概念的存在不构成自动开发承诺。
