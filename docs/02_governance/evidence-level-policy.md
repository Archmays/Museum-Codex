# 证据等级政策

## 等级含义

证据等级描述关系的认识论类别，不是“重要性”或一个可相加分数。

### A｜明确事实关系

可定位的师生、合作、工作室、成员记录；书信或本人陈述；可靠研究确认的直接接触。A 级必须说明直接性来自何处。第三方作者仅提出“可能受影响”通常不足以成为 `explicitly_influenced_by`。

### B｜有依据的历史语境

同一团体、学院、重要展览、地点/时期或有明确历史联系的技法。B 级证明共享语境，不证明个人接触或因果影响。

### C｜策展比较或计算相似

共享题材/技法/材料、学术比较、跨文化策展框架、色彩/构图/嵌入相似。C 级文案不得使用“影响了”“师承”或“直接导致”。

## 置信与强度

`evidence_level`、`evidence_confidence`、`historical_relationship_strength`、`computational_similarity`、`curatorial_relevance` 分开评估。A/B/C 不由置信阈值自动产生；证据较弱的 A 类候选可以低置信并停留 `disputed`。

每个数值字段必须有定义、生成者/算法、版本、范围和 `null` 语义。缺乏可解释度量时用 `null`，不得填 0.5 伪装精确。

## 升降级

- 新 Evidence 可触发人工重新分类；保存前后状态和理由。
- 算法 C 级不能自动升级为 B/A，也不能改名为影响关系。
- B 级共现即使时间重叠，也不能无接触证据升级为 A。
- 发现反证后进入 `disputed`；不是简单降低线条透明度。

## 展示文案

| 等级 | 允许示例 | 禁止示例 |
|---|---|---|
| A | “档案记录 X 曾在 Y 的工作室学习” | 没有定位的“公认师承” |
| B | “两人于同一时期在巴黎活动；尚无直接接触证据” | “因此 X 影响 Y” |
| C | “本展比较两作对光线的处理” | “相似构图证明借鉴” |

公开界面必须显示等级解释和来源入口，且允许用户关闭 C 级关系。
