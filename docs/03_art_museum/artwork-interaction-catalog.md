# 作品互动候选目录

互动按学习目标选择，不承诺每幅作品全部具备。

| 互动 | 学习目标 | 启用条件 | 主要风险/替代 |
|---|---|---|---|
| 局部放大镜 | 观察技法/细节 | 高分图与 crop 权利、移动性能 | 预选细节图 + 描述 |
| 细节猎人 | 主动观察 | 可验证细节、非纯记忆奖励 | 线性问题列表 |
| 双画对照 | 比较构图/材料/题材 | 两图权利和可比问题 | 并列 caption/表格 |
| 早期与晚期 | 观察变化 | 日期与归属可靠、避免目的论 | 时间范围说明 |
| 构图聚光灯 | 理解视觉组织 | overlay 变换可解释 | 静态标注图/文字 |
| 调色实验室 | 理解配色关系 | 明确为模拟，不改写原作 | 色板文字/数值 |
| 作品旅行地图 | provenance/展览史 | 地点时间 Claim 完整 | 事件列表 |
| 年代排序 | 时间推理 | 日期精度可比 | 列表/可跳过 |
| 策展人挑战 | 形成并解释选择 | 评价标准透明、不排名艺术 | 可访问表单 |
| 影响显微镜 | 区分 A/B/C | 关系证据可展开 | 证据卡片 |
| 修复前后 | 认识保存与研究 | 同源影像、技术说明和权利 | 并列静态图 |
| 红外/X 光/底稿层 | 看制作过程 | 正式技术影像与解释 | 预选帧 + 描述 |
| 观察后说明 | 减少先验提示 | 可随时跳过、屏幕阅读器顺序合理 | 默认展开设置 |
| 跨文化同题材 | 比较语境而非优劣 | 双方学科来源、避免同质化 | 策展文章/表格 |

## 通用门槛

每个启用实例记录作品 IDs、目标、Evidence、媒体权利、变换、适龄/语言、无障碍替代、性能预算、审核者和版本。互动结果不自动创建历史 Claim；用户操作不上传或收集，除非未来另有隐私 ADR。
