# 艺术星海交互规格（MUSEUM-04 正式 profile）

## 探索目标

星海帮助用户提出“这条关系是什么、证据在哪里、还有哪些替代路径”，不是展示最密集或最有地位的人。图谱只是固定 release 的一个视图。

## 初始状态

- 不加载/显示全部关系线；节点默认等大，显示少量代表性标签。
- 提供中英文/别名搜索、Graph/Artist list/Relationship table 切换，以及关系类型、A/B/C、时期、地区/传统和语境类型筛选。“算法相似”显示为 disabled/off，并说明当前 release 不含算法相似。
- 图例声明颜色、线型、透明度、线宽与距离的含义/非含义。

## 选择艺术家后

1. 以所选节点为焦点，展开一跳局部关系；相关节点高亮，无关节点降低亮度但不消失。
2. 边以类型颜色、A/B/C 线型和可访问标签显示；点击/键盘选择打开关系解释。
3. 解释面板显示端点、关系类型、范围、等级、置信、策展说明、Claim/Evidence/Source、争议和 release。
4. 用户可返回全局或关闭某类边。MUSEUM-04 不设置 AB 端点，不实现 AB 路径。

## 状态与 URL

可分享状态只包含 `release`、`focus`、`relation`、`types`、`level`、`period`、`region` 与 `view`，不嵌入说明文本、原始 Claim、未审核 ID 或用户历史。无效/撤回 ID 显示明确说明和可用替代。布局坐标在构建期以固定 seed/version/hash 预计算；坐标不是事实字段，运行时无随机或持续力导向。

## 移动与无障碍

移动端默认更小邻域、底部详情面板和更少标签；不要求悬停。键盘用 Tab、Enter/Space、Escape、Arrow、Home/End 完成筛选、焦点、视图与面板操作，并恢复可见焦点；屏幕阅读器使用同步实体列表与关系表，状态变化通过 live region 摘要。`prefers-reduced-motion` 禁用持续物理动画；forced colors、低带宽或 WebGL 失败自动进入等价列表并保留当前状态，不初始化或继续运行 Sigma。

## 性能门槛（MUSEUM-04 定量）

受控 lab 硬门禁覆盖 `390×844 / 4× CPU / Fast 4G`、`360×800 / 6× CPU`、`1366×768` 与 `1440×900`，记录多次运行的 median/p95、环境、FPS、heap、CLS/long tasks 与 gzip，且不得称为真实用户 p75。当前 12/36 图的移动/桌面 first interactive 目标分别为 ≤2.5 s/≤1.5 s；节点选择和键盘 p95 ≤100 ms；筛选/关系详情 p95 ≤200 ms；移动/桌面 FPS p95 ≥30/45；移动 heap ≤150 MB。1k/5k 使用实际 renderer 且移动 ≤5 s；10k/60k 不作全图首屏；50k/300k 的移动全量请求必须安全拒绝。实机不可用时记录 `not_available`，不伪称通过。若不达标，优先分片、预计算、减少可见 labels、局部邻域、Worker 或技术重评，不减少证据字段。

## 禁止语义

不显示“影响力大小”节点；不以中心位置表示地位；不把 C 级边表述为历史影响；不把过滤后的缺失解释成不存在关系。MUSEUM-04 只公开 36 条 C 级策展比较边、0 条 A/B 边和 0 条算法边；媒体仅使用 31 件作品通过 MUSEUM-03C 门禁的 242 个本地衍生图，另 13 件作品保持明确无图状态。媒体可用性不得映射为节点地位、关系强度或艺术价值。
