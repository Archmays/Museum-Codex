# 艺术家选择框架

本文件定义未来试点选择方法，不指定首批 12 人。

## 硬门槛

候选必须同时满足：

1. 身份稳定且明确为个人；
2. 死亡状态确认，出生/死亡资料有可靠 Claim；
3. 至少有可核验作品或正式艺术史记录；
4. 至少约 4 件作品具有可行的对象级展示权利路径，或有合理的较小配额说明；
5. 来源不只依赖 Tier 3；
6. 不因算法猜测创建身份；
7. 争议归属和别名可被 schema 正确表达。

不满足硬门槛者保留研究队列，不进入正式数据。匿名作者、工作室或群体可成为展览对象，但不占用“已故艺术家个人”名额。

## 组合维度

对通过硬门槛的候选做组合优化，而非给艺术地位排名：

| 维度 | 审查问题 |
|---|---|
| 地区与传统 | 是否避免单一区域/单一正典？跨文化比较是否有合适证据？ |
| 时间 | 是否覆盖不同历史区段，同时保持可解释连接？ |
| 性别与被忽视群体 | 来源缺口是否被误当成“不重要”？是否需更谨慎研究投入？ |
| 媒介/材料/题材 | 是否支持多种观察和互动，而不强迫每人同质化？ |
| 作品/来源可用性 | 是否有稳定馆藏页、权威身份、引用定位？ |
| 图片许可 | 能否逐对象确认并公开展示，是否有高分辨率/IIIF？ |
| 关系可解释性 | 是否有 A/B/C 的平衡和可读路径，而非制造边？ |
| 公众学习价值 | 是否能形成观察、比较、时空与证据问题？ |

## 评分用途

可以用 `0–3` 记录每一维的资料准备度与组合贡献，但分数只帮助形成候选组合，不代表艺术价值。需同时列证据、缺口和权利风险；任何硬门槛不能被总分抵消。

## 选择流程

建立宽候选池 → 身份/死亡硬门槛 → 来源与媒体权利预检 → 组合情景（地区/时间/性别/媒介）→ 关系与学习路径可行性 → 偏差审查 → 用户确认 → 才进入 MUSEUM-03 采集。公布纳入标准与局限，不声称代表全部艺术史。
